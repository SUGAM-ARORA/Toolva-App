# Toolva Mobile

A comprehensive mobile application that serves as a one-stop directory for AI tools and services. Built with React Native and Expo.

## Features

- 📱 Cross-platform support (iOS, Android, Web)
- 🎨 Dynamic theme switching (Light/Dark mode)
- 🔍 Advanced search functionality
- ⚡ Interactive AI tools playground
- 💾 Save favorite tools
- 🔒 Secure authentication
- 📊 Real-time tool ratings and reviews

## Getting Started

### Prerequisites

- Node.js (v18 or newer)
- npm or yarn
- Expo CLI

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/toolva-mobile.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

## Project Structure

```

│   └── theme/             # Theme configuration

```
